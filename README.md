# task-planner
A task planning application designed to manage, edit, and organize daily tasks efficiently.

## Features
- Add new tasks
- Edit tasks
- Delete tasks
- Save tasks

## Technologies Used
- HTML
- CSS
- JavaScript
- Python (Flask)
- SQLite

## Purpose
This project was created as a learning project to understand how a full to-do app works.

## How to Run
1. Clone the repository
2. Run the backend server
3. Open the app in your browser
